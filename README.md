# APP - Cadastro Usuario Mobile
## Descrição:
 Aplicativo desenvolvido no Android Studio com objetivo de implementar o cadastro de usuários e aprender conceitos de construção de Interface no Android.
 
 O app permite a entrada de dados básicos (nome e e-mail) e exibição dessas informações em uma lista dinâmica utilizando RecyclerView.


### 🛠️ Tecnologias Utilizadas
 * Java 
 * Android Studio
 * XML para Layout 
 * Git/Github:versionamento de código

## 🧠 Lógica Implementada

Foi criado um método responsável por capturar os dados inseridos pelo usuário através dos campos de entrada(Nome, E-mail)

## 📋 Exibição de Dados com RecyclerView

Para gerenciar a listagem de usuários, foi criado o componente:

🔧 AdapterUser

Responsável por: Criar os itens da lista,
Fazer o bind (vinculação) dos dados aos elementos visuais,
Gerenciar a exibição dentro da RecyclerView

## Exemplo de fluxo:

Usuário preenche nome e e-mail

Sistema captura os dados

Exibe lista de usuários
