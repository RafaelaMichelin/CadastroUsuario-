# APP - Cadastro Usuario Mobile
## Descrição:
### Aplicativo desenvolvido no Android Studio com objetivo de implementar o cadastro de usuários e aprender conceitos da Interface


### 🛠️ Tecnologias Utilizadas
 * Java 
 * Android Studio
 * XML para Layout 
 * Git/Github:versionamento

## 🧠 Lógica Implementada

Foi criado um método responsável por capturar os dados inseridos pelo usuário nos campos de nome e e-mail através de inputs na interface.

### Exemplo de fluxo:

Usuário preenche nome e e-mail

Sistema captura os dados

(Opcional: validação ou envio)
